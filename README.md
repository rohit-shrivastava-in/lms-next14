## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


replace keys in .env
```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=###########################################
CLERK_SECRET_KEY=################################################
```
